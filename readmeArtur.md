# SecureMail

Приложение для обмена письмами между аккаунтами с вложениями. Реализованы вход, гостевой режим через кнопку «Пропустить», список писем, просмотр письма, отправка сообщений и файлов, бургер-меню и настройки. Технологии: React 18 + Vite, TypeScript, Tailwind, React Router, Zustand, React Hook Form, axios; Backend на Node.js + Express, Prisma + SQLite, JWT, bcrypt, multer, helmet, rate-limit.

Запуск: установить Node.js, в папках frontend и backend выполнить `npm install`. Запустить бэкенд `npm run dev` в папке backend, затем фронтенд `npm run dev` в папке frontend. Логины: Sam4k/1234 (email sam@example.test) и artur/1234 (email artur@example.test). На странице входа есть кнопка «Пропустить» для гостевого входа.

Отправка: на главной странице заполнить получателя (логин или email), тему, текст и прикрепить файлы. Сообщение появится в списке у получателя, вложения доступны по ссылкам. Безопасность: пароли хешируются, доступ по токенам, файлы ограничены по типам и размеру, включены защитные заголовки и ограничение запросов.

Тестирование: зайти под Sam4k, отправить сообщение artur, затем зайти под artur и открыть входящие. Проверить открытие вложений. Докер-режим доступен через docker-compose. Seed создаёт два аккаунта и примерные сообщения.
